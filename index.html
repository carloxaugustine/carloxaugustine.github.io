HOTtake / SOLmate

Python + Elasticsearch pipeline for NASA's Mars rover thermal team. Flags weird heater preheat events, surfaces them on dashboards, and cuts manual review time ~60%. Used by flight thermal engineers.

Closed-Loop Thermal Control Sim

Modeled rover-like thermal zones and built a PID-style controller in MATLAB/Simulink to hold components inside safe temperature limits. Produced tuning curves and stability checks for engineers.

Telemetry Parser (C++)

C++ tool that walks mission-style thermal telemetry records, extracts key fields, and raises out-of-family flags with minimal heap churn. Built like flight software: predictable, testable.

3-DOF Arm Kinematics Sandbox
